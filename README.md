#  SkillCraft Technology – Data Science Internship

## Task 01: Population Distribution



##  Project Overview

As part of the **SkillCraft Technology Data Science Internship**, I successfully completed **Task 01: Population Distribution**. The objective of this project was to analyze population-related data and create meaningful visualizations using **Bar Charts** and **Histograms** to understand demographic distributions and identify patterns within the dataset.

The entire project was developed and executed in **Google Colab**, where Python libraries were used to load, analyze, and visualize the data. Through this task, I gained practical experience in data analysis, exploratory data analysis (EDA), and data visualization techniques that are widely used in real-world Data Science projects.



##  Objective

The primary objectives of this project were:

- Analyze population-related data.
- Understand demographic distribution patterns.
- Create Bar Charts for categorical data visualization.
- Create Histograms for numerical data distribution analysis.
- Perform basic Exploratory Data Analysis (EDA).
- Extract meaningful insights from the dataset.
- Develop practical Data Science skills using Python.
